# skillrepos

Monorepo for reusable Codex skills.

## Skills
- `skills/large-file-split-migration`
- `high-bar-project-scoring`

## Sync workflow
1. Update source skill in your workspace.
2. Sync the entire skill folder into this repo path (not only `SKILL.md`).
3. Update `README.md` skill list.
4. Commit and push.
